Course Curriculum

Introduction

Welcome to the Web Development course. This course is designed to help students learn the fundamentals of HTML, CSS, and JavaScript to build professional websites. By the end of the course, you’ll be able to create your own web projects! Don’t forget to bring your enthusiasm & creativity 😃

Course Objectives

  • Understand the structure of HTML documents
  • Use CSS for styling
  • Implement basic JavaScript for interactivity
  • Build and publish a responsive website

Course Modules

  1. HTML Basics
    • Headings, Paragraphs, and Formatting
    • Lists (Ordered, Unordered, Description)
    • Links & Images
  2. CSS Styling
    • Selectors and Properties
    • Colors, Fonts, and Backgrounds
    • Box Model & Layout
  3. JavaScript Essentials
    • Variables and Data Types
    • Functions and Events
    • DOM Manipulation

Course Duration

The course will run for 12 weeks (3 months), with 2 classes per week. Each session will include both theory and practical exercises.

Grading Criteria

Assignments
Weekly hands-on tasks (40%)
Midterm Project
A small website based on learned concepts (30%)
Final Project
Full web application submission (30%)
